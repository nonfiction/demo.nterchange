nterchange Demo

  1. Content - Click the 'Content' tab and create a text asset. We'll use this later.
  2. Surf-to-edit - Add, remove and re-order content on the about page.
  3. Page Content - Add the text asset from step one to the sidebar and main content area. Edit the text to see changes happen to all instances.
  4. Custom assets - Edit a bio, add an image (80x80) and change the description.
  5. Dynamic Asset Lists - Manage the news items. Create an item for today and see it pulled onto the Home and news pages.
  6. Pages
    • Create a new page that is Visible and Active, and see it show up in the navigation.
    • View page tree in the Site Admin tab.
  7. Versions - View an asset with many previous versions.
  8. Timed Content. Surf-to-edit the Contact page, and set the "Timed Start" of the first piece of content.
  9. Custom asset behaviour - Set the release date for a news item one minute in the future. View the list on the news page or Home page.
